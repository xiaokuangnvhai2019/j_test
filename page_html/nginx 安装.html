nginx 安装

一、官网下载软件

http://nginx.org/en/download.html

这里我下载的是1.12.2版本   

 

二、上传到Linux服务器

  • rz命令，或者Xshell软件，直接拖到对应目录

  • 解压目录  tar -xvf nginx-1.12.2.tar.gz

 

三、安装依赖

  • 依赖：pcre   openssl   zlib  gcc

  • 可通过下载包进行安装，也可通过yum 安装，这里我们通过yum安装

  • yum -y install  gcc zlib zlib-devel pcre-devel openssl openssl-devel

  • 安装检查：比如 pcre-config --version

 

四、安装Nginx

  • 进入到解压目录，执行./configure

  • make && make install

  • /usr/local/nginx 默认安装路径

  • 安装路径下的sbin文件下，./nginx -v   查看安装版本

  • 安装路径下的sbin文件下，执行./nginx，启动程序

 

 

五、防火墙设置

  • 查看开放端口：firewall-cmd --list-all

  • 增加开放端口：firewall-cmd --add-port=80/tcp --permanent 

  • 重启防火墙：firewall-cmd -reload

  • 备注：firewall-cmd --add-service=http -permanent 开发服务
