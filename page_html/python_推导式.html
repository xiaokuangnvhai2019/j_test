1、列表生成式

列表生成式即List Comprehensions，是Python内置的非常简单却强大的可以用来创建list的生成式列表生成式

#非列表生成式写法

list1=[]

for i in range(1,10):

    reult=i*i

    list1.append(reult)

print(list1)



#列表生成式

list2=[i*i  for i in range(1,10)]

print(list2)

2、三元表达式

三元表达式主要由三部分组成循环，条件判断，操作

 

例子：

#非三元式

list1=[]

for i in range(1,20):

    if i%2:

       list1.append(str(i).zfill(3))

print(list1)



#三元式

list2=[str(i).zfill(4) for i in range(1,20) if i%2]

print(list2)
