五、生成公钥和秘钥

cd /root/.ssh



ssh-keygen -t rsa -C "mghxy123@163.com" #邮箱是Git用户邮箱



备注：一路回车即可，无需输入密码





路径下生成2个文件：id_rsa 和 id_rsa.pub



id_rsa   秘钥



id_rsa.pub 公钥
